# Математический аппарат

## Дискретная математика

Используется для представления алфавита как конечного множества символов.

Пример:

А = {А, Б, В, ..., Я}

Каждому символу сопоставляется порядковый номер.

---

## Модульная арифметика

Используется в алгоритмах Цезаря и Виженера.

Шифрование в шифре Цезаря:

E(x) = (x + k) mod n

где:

* x — номер символа;
* k — ключ;
* n — размер алфавита.

Расшифрование:

D(x) = (x − k) mod n

---

## Математическая статистика

Используется для частотного анализа.

Относительная частота символа:

p = nᵢ / N

где:

* nᵢ — количество появлений символа;
* N — общее количество символов в тексте.

---

## Элементы теории информации

Используются для анализа распределения частот символов в текстах естественного языка.

# Функциональные требования

## Работа с файлами

Программа должна обеспечивать:

* открытие текстовых файлов;
* чтение содержимого файла;
* отображение текста;
* сохранение результата в новый файл по запросу пользователя.

---

## Шифр Цезаря

Поддерживаемые операции:

* шифрование текста;
* дешифрование текста;
* ввод числового ключа.

---

## Шифр Виженера

Поддерживаемые операции:

* шифрование текста;
* дешифрование текста;
* ввод ключевого слова.

---

## Шифр Полибия

Поддерживаемые операции:

* шифрование текста;
* дешифрование текста.

---

## Частотный анализ

Поддерживаемые операции:

* подсчет частоты символов;
* вычисление относительных частот;
* определение вероятного ключа шифра Цезаря;
* автоматическая попытка дешифрования текста.
